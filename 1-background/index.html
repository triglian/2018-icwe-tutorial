Web-based Presentations

Advantages

  • Open format using HTML5
  • Declarative formatting with CSS
  • Play slides in any compatible Web browser
  • Responsive to adapt to different screen sizes/devices
  • Programmable interactive slides

Disadvantages

  • Limited WYSIWYG editors (improving)
  • Difficult to reuse individual slides
  • Too many frameworks to choose from
